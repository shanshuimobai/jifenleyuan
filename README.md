# 🌟 积分乐园 - 孩子学习劳动积分系统

> 🌐 **在线访问**: https://shanshuimobai.github.io/jifenleyuan/

一个运行在浏览器中的孩子学习积分管理应用，支持 **iPad / iPhone / 安卓平板**，无需安装任何软件。

## ✨ 功能

- ✅ **完成任务赚积分** — 点击卡片即可完成，获得积分
- 🎁 **积分兑换奖励** — 攒够积分兑换奖励，有确认环节防误触
- 🍅 **番茄时钟** — 首页内置番茄专注计时器，培养专注力
- ⭐ **动态积极评价** — 每次完成任务随机出现不同的鼓励语
- 📊 **每日限完成一次** — 每个任务每天只能完成一次
- 📱 **PWA 支持** — 添加到主屏幕后全屏运行，像原生 App

## 🚀 使用

1. 在浏览器打开 https://shanshuimobai.github.io/jifenleyuan/
2. 输入孩子名字和 4 位家长密码
3. 开始使用！

### 添加到主屏幕（iOS）

用 Safari 打开 → 分享按钮 → 添加到主屏幕
